This plugin allows you to create documentation (no sweat) on your WordPress Powered Website. No Setup required.

== Description ==
No Sweat Documentation is an easy-to-use plugin that allows you to create documentation (with a custom post type) on your WordPress Website. This plugin is a solution for creating quick documentation pages that are simple and can be for whatever reason you need.

Notable Features:
<ul>
<li>A [all_docs] shortcode that makes it easy to display child pages for that documentation (Inspired by http://make.wordpress.org/docs/theme-developer-handbook/)</li>
<li>A new user role called Documentation Master. This role allows a user to upload files and only edit documentation.</li>
</ul>





== Installation ==
Easy Installation

One of the easiest ways to install the plugin is by going into your dashboard, find the plugins menu, and click Add New. Then, search for No Sweat Documentation. Once you have found the plugin, you may read the details and then click Install Now.

After that, head to the Documentation Admin Menu and start writing away!
